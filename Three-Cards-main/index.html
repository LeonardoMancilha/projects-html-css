Html

Ela é utilizada como marcação para desenvolver páginas e documentos eletrônicos para a internet. Isso significa que ela garante a formatação ideal para sites.

Css

É usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site.

Javascript

É uma Linguagem de programação que permite a você implementar itens complexos em páginas web. É uma linguagem criada para servir ao usuário.
